# dEmOn Speed Dial - Chrome 扩展

这是一个专门定制的 Chrome / Edge 浏览器扩展，用于将新标签页替换为设计高雅的 Speed Dial 快速拨号界面。它完全从您的个人导航网站（Cloudflare Pages 后端）同步数据，支持快速拨号展示、自定义收藏、实时搜索、管理员登录、添加/编辑/删除以及拖拽排序等功能。

## 功能特点

- **新标签页接管**：每天打开新标签页时，展示优雅的快速拨号卡片。
- **数据自动同步**：拉取后台分类和链接数据，完全与网页端同步。
- **拖拽排序**：在编辑模式下，可以像网页版一样，通过拖拽手柄重新排列链接。
- **本地收藏**：可以在卡片右上角标记常用链接为收藏（星标），数据存储于本地扩展中。
- **安全沙箱 (CSP)**：全新升级，完全符合 Chrome MV3 内容安全策略。

## 安装步骤

1. 打开 Chrome 浏览器，访问：`chrome://extensions/`
2. 在右上角开启 **"开发者模式" (Developer mode)**。
3. 点击左上角的 **"加载已解压的扩展程序" (Load unpacked)**。
4. 选择当前项目目录下的 `extension` 文件夹（即包含 `manifest.json` 的目录）。
5. 加载成功后，新标签页将已被成功接管！

## 配置与使用

- **首次使用**：默认 API 域名已配置为 `https://nav.ipanic.bond`。
- **更改 API 域名**：在新标签页右上角点击 **"⚙️ 设置"**，可输入您自定义的后端 API 接口地址，测试连接无误后保存即可。
- **内容管理**：点击右上角 **"✏️ 编辑"** 并输入您的管理密码，即可进入编辑模式直接操作分类及链接（添加、编辑、删除、拖拽排序）。完成操作后点击右上角 **"✅ 完成"** 即可退出编辑状态。
